

Requires Python 3.10+.

### macOS
Double-click `desktop/run_mac.command`. The first run installs PySide6. macOS may ask you to approve opening the script; Control-click and choose Open if necessary.

### Windows
Double-click `desktop/run_windows.bat`.

### Linux
Run `./desktop/run_linux.sh`. On some Wayland compositors, always-on-top transparent windows may behave differently; X11 generally works best.

You can also run:

```bash
python3 desktop/desktop_cats.py --count 7 --size 150 --speed 55
```

Drag a cat with the left mouse button. Click to pause/resume, double-click to reverse direction, or right-click for the menu and Quit command.

## Cutout note

The images are lightweight sticker-style photo cutouts optimized for small animated sprites. The original photos are untouched.
